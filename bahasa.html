Bahasa pemrograman


Bahasa pemrograman yaitu sebuah rangkaian kata berbentuk instruksi atau perintah-perintah yang terdiri dari satu atau beberapa baris yang dimengerti oleh komputer. Dalam membuat aplikasi tertentu, digunakan juga bahasa pemrograman yang sesuai dengan kebutuhan aplikasi maka dari itu, seorang developer wajib menguasai atau memahami berbagai bahasa pemrograman tersebut.


Apa itu Bahasa Pemograman ?


Bahasa program adalah sekumpulan instruksi yang diberikan kepada komputer untuk dapat melaksanakan tugas-tugas tertentu dalam menyelesaikan suatu permasalahan. Bahasa program berfungsi untuk memerintah komputer agar dapat mengolah data sesuai dengan langkah - langkah penyelesaian yang telah ditentukan oleh programmer. Bahasa ini memungkinkan seorang programmer untuk menentukan secara persis data mana yang akan diolah oleh komputer, bagaimana data ini akan disimpan/diteruskan, dan jenis langkah apa yang akan diambil dalam berbagai situasi secara persis.

  • python

    python adalah salah satu bahasa yang cukup populer belakangan ini. bahasa ini cukup digemari orang-orang yang ingin berkutik pada data. bahasa ini cocok digunakan oleh pemula karena strukturnya tidak serumit bahasa pemrograman yang lain dan mudah untuk dipahami. Biasanya, bahasa python banyak diterapkan pada pengembangan dan peningkatan IoT, pembuatan sotware workflow.

  • Java

    Java adalah bahasa pemrograman yang berjalan di berbagai platform, mulai dari komputer pribadi hingga perangkat seluler. Bahasa pemrograman ini awalnya dikembangkan pada tahun 1991 di The Green Project dari Sun Microsystems dan kemudian dirilis oleh Oracle.Nama Java sendiri dipilih karena salah satu pendiri Java, James Gosling, sangat menyukai kopi murni yang digiling langsung di grinder (kopi Tubruk). Ini adalah jenis kopi berasal dari pulau Jawa.

  • C

    Bahasa pemrograman C adalah bahasa pemrograman komputer untuk keperluan umum. bahasa ini ditulis oleh Dennis Ritchie pada tahun 1972 di Bell Telephone Laboratories untuk sistem operasi Unix. Secara desain, fitur C secara jelas mencerminkan kemampuan CPU target. Bahasa C banyak dikembangkan dan digunakan untuk pemrograman sistem komputer jaringan, dan pengembangan aplikasi atau perangkat lunak. Sejak tahun 2000, bahasa C secara konsisten dapat menepati peringkat dua bahasa teratas indeks TIOBE , untuk bahasa pemrograman paling populer.

  • C++

    C++ adalah bahasa pemrograman komputer yang dikembangkan oleh Bjarne Stroustrup dan merupakan evolusi dari bahasa pemrograman C yang dikembangkan oleh Bell Labs (Dennis Ritchie). Bjarne Stroustrup dari Bell Labs pertama kali mengembangkan C++ pada awal 1980-an. Sistem perampingan dan dukungan pemrograman tingkat rendah (pengkodean tingkat rendah) dikembangkan untuk mendukung fitur-fitur C++. Salah satu perbedaan paling mendasar dari bahasa C adalah dukungannya untuk pemrograman berorientasi objek.

  • Kotlin

    Kotlin adalah bahasa pemrograman yang diketik secara statis yang berjalan di Java Virtual Machine atau menggunakan kompiler LLVM yang dapat dikompilasi ke kode sumber JavaScript. Pengembang utamanya adalah tim programmer JetBrains. Kotlin dirancang untuk bekerja sama dengan kode bahasa Java dan mengandalkan kode bahasa Java di perpustakaan kelas Java yang ada seperti Berbagai framework Java yang ada.


Scripting Languages

JavaScript

Lebih dari 95% situs web yang menggunakan JavaScript membuat namanya sangat populer di kalangan pengembang. Pengembang JavaScript biasanya menggunakannya sebagai ujung depan, berbeda dengan Java dan Python, yang digunakan pengembang sebagai ujung belakang. Framework yang ada pada JavaScript juga banyak digunakan untuk mendukung pengembangan Internet of Things (IoT) pada perangkat mobile.



PhP

PHP adalah bahasa pemrograman yang biasa digunakan untuk pembuatan dan pengembangan web. PHP adalah singkatan dari PHP: Hypertext Preprocessor. Php cocok untuk dipelajari jika kamu ingin menggeluti dunia website. Php adalah dasar yang paling penting untuk mengembangkan sebuah website. Php memiliki struktur yang mudah dan tidak rumit.



Perl

Perl adalah bahasa pemrograman tujuan umum yang awalnya dikembangkan pada mesin Unix oleh Larry Wall. Perl pertama kali dirilis pada 18 Desember 1987, saat Perl 1 dirilis. Dalam versi yang lebih baru, Perl juga tersedia untuk berbagai varian sistem operasi Unix (SunOS, Linux, BSD, HP-UX). Dua fitur utama Perl adalah pengolah kata dan jalan pintas untuk memecahkan masalah umum.



VbScript

VBScript atau Full Visual Basic Scripting Edition adalah bahasa skrip yang diinterpretasikan runtime yang dikembangkan oleh Microsoft Corporation pada tahun 1996, umumnya digunakan di browser web Internet Explorer (untuk Internet Explorer 3.0 dan yang lebih baru). Aku bisa melakukan itu). VBScript adalah bahasa scripting yang berasal dari bahasa pemrograman Visual Basic for Applications (VBA) yang digunakan oleh Microsoft Office dan beberapa platform pengembangan Microsoft lainnya. VBScript menghapus beberapa fitur VBA, seperti: B. Kemampuan file I/O dan akses langsung ke sistem operasi menyediakan platform yang aman untuk mengembangkan aplikasi berbasis Web menggunakan platform Active Server Pages (ASP).
